# UniPlanner

Academic planning system for degree tracking and course scheduling.

## Quick Start

### Setup (First Time)
```bash
./setup.sh
```

### Start Servers
```bash
./start_servers.sh
```

Or use Python runner:
```bash
python run.py
```

### Stop Servers
```bash
./stop_servers.sh
```

## Tech Stack

- **Frontend**: React.js with Material-UI
- **Backend**: Django REST Framework
- **Database**: SQLite
